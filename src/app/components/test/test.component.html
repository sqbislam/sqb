Basics of Computer Vision and the Deep Learning fundamentals behind it.

Computer Vision deals with the idea of computers being able to visually see the real world and interpret information about it just like we humans do. In simple terms, it gives eyes to the computer system. This, if you think about it is not an easy task. For instance, us as human beings have been visualising the world from when we were born, and hence we have an inherent understanding of it. But Computers for instance, cannot see images but a matrix of integer values, of each individual pixel. Each pixel value may range from 0-255 and have three channels for color (RGB). This is called the semantic gap. We have given the image a semantic meaning based on its appearance which the computer is not capable of grasping. Here is a visulaisation.

So it is a challenge to make computers percieve the world, and interpret it using just these numbers. Even if you want to hard code an alogrithm which figures out the different edges or corners to identify the the picture of a cat. It will work for this picture only. But if you give it a new picure of a cat, it will fail to identify it since even the slightest variation of the image, changes all the pixel values. We need to make a model that generalizes, and is robust enough to identify any image of a cat, from any angle and orientation. How do we do this?

This is where Deep Learning or Neural Nets comes in. A network of layers of computation to create a model that is remarkably able to achieve the desired outcome. The terminology may seem overwhelming at first, I myself was always scared of the term 'Deep Learning or Neural networks' but the fundamentals are pretty simple and intriguing to say the least.
